debug.js
Easy Debugging w/o F12
Debugger of JavaScript, by JavaScript, for JavaScript
Logging, Time Test, Element Inspection, Screen Measure, and more
// Usage

Embed the debugger into your web page
Include the script file. 
<script src="debug.js"></script>
That's it!

The debug window will be automatically generated and is draggable.

⚠The debug window is hidden by default setting.
You can switch show/hide by F2 key.

Logging
Put log messages anywhere you want.
log(message);
e.g.,)
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <script src="debug.js"></script>
  <script>
    window.onload = function() {
      log('The window\'s load event fired.');
    }

    function foo() {
      ...
      log('bla bla bla');
      ...
      if (error) {
        log.e('error! val=' + val);
      }
      ...
    };
    ...
  </script>
</head>
<body>
  Hello, World!<br>
  ...
</body>
</html>

// Methods
Name Argument Description
log message string Output standard log
log.d message string Output debug log
log.i message string Output information log
log.w message string Output warning log
log.e message string Output error log
log.s message string Output system log
log.p arg1: variable
arg2: recursion level limit(opt)
arg3: message string(opt) 		Output object dump
log.t arg1: message string
arg2: timer name(opt) 		Output log w/ elapsed time 
log.t('foo', 'timer1')
-> 00:00:12.345 foo

log.t('foo')
-> 00:00:12.345 foo
default timer name will be used in case no name is given to the timer name.
log.res message string Output result log
> message
log.res.err message string Output error result log
> message
log.stack Output stack trace
log.clear Clear log message
time.start arg1: timer name(opt)
arg2: message(opt) 		Start a timer 
time.start()
-> timer0: timer started
default timer name will be used in case no arguments are specified.

time.start('timer1')
-> timer1: timer started

time.start('timer1', 'Timer Start: %n')
-> Timer Start: timer1

time.start('timer1', null)
-> no output log (only start the timer)
time.split arg1: timer name(opt)
arg2: message(opt) 		Print elapsed time of the timer 
time.split()
-> timer0: 00:01:02.345 (⊿00:01:02.345)
default timer name will be used in case no arguments are specified.

time.split('timer1')
-> timer1: 00:01:02.345 (⊿00:01:02.345)

time.split('timer1')
-> timer1: 00:01:03.345 (⊿00:00:01.000)

time.split('timer1', '[%n] epalsed time: %t (%lt)')
-> [timer1] elapsed time: 00:01:03.345 (⊿00:00:01.000)

time.split('timer1', null)
-> no output log (only split the timer)
time.end arg1: timer name(opt)
arg2: message(opt) 		Print elapsed time of the timer and clear. 
time.end()
-> timer1: 00:01:02.345
default timer name will be used in case no arguments are specified.

time.end('timer1')
-> timer1: 00:01:02.345

time.end('timer1')
-> timer1: 00:01:03.345 (⊿00:00:01.000)

time.end('timer1', '[%n] epalsed time: %t (%lt)')
-> [timer1] elapsed time: 00:01:03.345 (⊿00:00:01.000)

time.end('timer1', null)
-> no output log (only clear the timer)
time.check timer name Get elapsed time of the timer. 
time.check()
-> A string like "00:01:23.456" will be returned.
default timer name will be used in case no arguments are specified.

time.check('timer1')
-> A string like "00:01:23.456" will be returned.

The debugger can perform the following calculation/conversion on the command-line.

Radix conversion decimal number
hexadecimal number
binary number 		Convert DEC / HEX / BIN to other radix base number. 
$ 128
->
DEC 128
HEX 0x80
BIN 1000 0000 (8 bits)

$ 0x20
->
HEX 0x20
DEC 32
BIN 10 0000 (6 bits)

->
BIN 1111 0000 (8 bits)
DEC 240
HEX 0xF0
Time calculation HH:MI:SS.sss + HH:MI:SS.sss
