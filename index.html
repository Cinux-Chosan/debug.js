debug.js - demo & description
Usage:

HTML
Include the script file.
<script src="debug.js"></script>

JavaScript
Put log messages anywhere you want.
log(<message>);

That's it!

The debug window will be automatically generated.
The window is draggable and you can switch hide/show by F2 key.

e.g.,) 
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <script src="debug.js"></script>
  <script>
    window.onload = function() {
      log('The window\'s load event fired.');
    }

    document.onkeydown = function(e) {
      log('KeyDown: keyCode = ' + e.keyCode);
    }

    document.onmousemove = function(e) {
      log('mouseMoved: x=' + e.clientX + ', y=' + e.clientY);
    };
    ...
  </script>
</head>
<body>
  Hello, World!<br>
  ...
</body>
</html>

var options = {
  'position': 'center',
  'showTimeStamp': false
};

Debug.init(options);

Methods:
Name Argument Description
log messag string Output standard log
log.e messag string Output error log
log.w messag string Output warning log
log.i messag string Output information log
log.d messag string Output debug log
log.v messag string Output verbose log
log.s messag string Output special log
log.p variable Output object dump
log.stk Output stack trace

Commands:
Name Arguments Description
cls Clear log message.
exit Close the debug window.
p variable(s) Print object dump.
help Displays available command list.
history Displays command history.
rgb RGB Color Value Convert RGB color values between HEX and DEC.

$ rgb #fff
-> 255 255 255

$ rgb 255 255 255
-> #fff
v Displays version info.
