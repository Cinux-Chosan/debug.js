debug.js demo & description
// Usage

HTML
Include the script file. Download 
<script src="debug.js"></script>

JavaScript
Put log messages anywhere you want.
log(<message>);
That's it!

The debug window will be automatically generated.
You can switch hide/show by F2 key and the window is draggable.

(note) The debug window is hidden by default setting.

e.g.,) 
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <script src="debug.js"></script>
  <script>
    window.onload = function() {
      log('The window\'s load event fired.');
    }

    function foo() {
      ...
      log('bla bla bla');
      ...
      if (error) {
        log.e('error! val=' + val);
      }
      ...
    };
    ...
  </script>
</head>
<body>
  Hello, World!<br>
  ...
</body>
</html>

// Methods
Name Argument Description
log messag string Output standard log
log.e messag string Output error log
log.w messag string Output warning log
log.i messag string Output information log
log.d messag string Output debug log
log.s messag string Output special log
log.p variable Output object dump
log.stack Output stack trace
log.clear Clear log message
time.start arg1: timer name
arg2: message(opt) 		Start a timer 
time.start('timer1')
-> timer1: timer started

time.start('timer1', 'Timer Start: %n')
-> Timer Start: timer1
time.split arg1: timer name
arg2: message(opt) 		Print elapsed time of the timer 
time.split('timer1')
-> timer1: 00:01:02.345

time.split('timer1')
-> timer1: 00:01:03.345 (⊿00:00:01.000)

time.split('timer1', '[%n] epalsed time: %t (%lt)')
-> [timer1] elapsed time: 00:01:03.345 (⊿00:00:01.000)
time.end arg1: timer name
arg2: message(opt) 		Print elapsed time of the timer and clear. 
time.end('timer1')
-> timer1: 00:01:02.345

time.end('timer1')
-> timer1: 00:01:03.345 (⊿00:00:01.000)

time.end('timer1', '[%n] epalsed time: %t (%lt)')
-> [timer1] elapsed time: 00:01:03.345 (⊿00:00:01.000)
dbg.countElements element selector Count number of elements. 
dbg.countElements('*');
-> Count all elements

dbg.countElements('#id');
-> Count the id's element
dbg.init options Initialize debug.js
