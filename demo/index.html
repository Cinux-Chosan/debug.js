debug.js - Log Message Demo

HTML:
include the files.
<link rel="stylesheet" href="debug.css" />
<script src="debug.js"></script>

set a messag area.
<div id="log" class="log"></div>

JavaScript:
output the log.
log(<message>);

(optional)
If you want to change the log settings, explicitly call initialization function in advance.
arguments: (<id of the message area>, <buffer size>, <show line numbers>)
Debug.init("log", 5, true);
