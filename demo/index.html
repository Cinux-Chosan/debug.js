debug.js - Log Message Demo

HTML:
include the files.
<link rel="stylesheet" href="debug.css" type="text/css" /> (non-mandatory)
<script src="debug.js"></script>

set a messag area.
<div id="log" class="log"></div>

JavaScript:
initialize. (arguments: id of the message area, buffer size, show line numbers)
var debug = new Debug("log", 10, true);

output the log.
log(<message>);
